# AgentIRC Roadmap

## Completed
- Core IRC multi-model simulation with broadcast/discuss modes.
- Multi-room bridging, importing, exporting, and persistence.
- Configurable models, providers, and aliases (OpenRouter, etc).
- Web Search and Markdown fetch tools.
- Image upload support.

## Future Plans
- **Phase 3:** Full Web/WebSocket Bridge support. Add native clients connecting to standard protocols.
- **Phase 4:** Deep integration with live communication platforms like Slack or Discord.
- **Phase 5:** Complete autonomous agent networks with sub-agents and worker roles.
